# How to get the API_KEY ?

### `Why is it required?`
To fetch movie data required for the web-app

## Follow these steps to get the API:
1. Go to [https://www.themoviedb.org](https://www.themoviedb.org)
2. Click on `Join TMDB` and create an account.
3. Click on `Account` and then go to `Settings`.
4. On the side menu, click on `API` and access the API_KEY along with other API information.

## Follow these steps to integrate the API_KEY to the web-app
1. Go to `netflix-clone/src/Routes/requests.js/` in the file directory and replace `process.env.API_KEY` with the `obtained API_KEY`.
